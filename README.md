# AR Flashcard Learning Platform

Interactive educational web application designed to help children learn alphabets using flashcards and engaging visuals.

---

## Features
- Interactive alphabet flashcards
- Simple and responsive UI
- Flask backend integration
- Educational learning interface
- Expandable AR/3D learning concept

---

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Open in browser:

```txt
http://127.0.0.1:5000
```

---

## Purpose

This project was created as part of a software engineering portfolio to demonstrate full-stack development concepts using Flask and frontend technologies.
